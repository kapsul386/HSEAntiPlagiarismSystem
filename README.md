
# HSE AntiPlagiarism System

Микросервисное приложение для приёма студенческих работ и автоматического определения плагиата.  
Проект выполнен в рамках дисциплины «Конструирование программного обеспечения», домашняя работа №3.

## Основные возможности

- Приём файлов с указанием студента и задания
- Генерация криптографического отпечатка содержимого (SHA-256)
- Проверка работ на плагиат по совпадению отпечатков
- Получение истории отчётов по заданию
- Развёртывание всех сервисов через Docker Compose

## Архитектура

Приложение состоит из трёх микросервисов:

| Сервис       | Назначение |
|-------------|------------|
| Gateway     | Принимает запросы, взаимодействует с другими сервисами и формирует общий результат |
| FileStoring | Сохраняет файл и отдаёт метаданные |
| Analysis    | Хранит историю работ и анализирует их на предмет плагиата |

### Логика взаимодействия

Клиент → Gateway → FileStoring  
→ Analysis  
← Gateway

## Логика определения плагиата

1. Получение файла и расчёт SHA-256 (`ContentFingerprint`)
2. Регистрация сдачи в Analysis
3. Поиск более ранней сдачи другого студента с тем же отпечатком

```json
{
  "isPlagiarism": true,
  "plagiarismSourceStudentId": "student-1"
}
````


## Запуск

```bash
docker compose build
docker compose up -d
```

Gateway доступен по адресу:

```
http://localhost:5000
```

Остановка:

```bash
docker compose down
```

## API тестирование

Коллекция Postman:

```
docs/AntiPlagiarismSystem.postman_collection.json
```

| Операция         | Метод | URL                           |
| ---------------- | ----- | ----------------------------- |
| Загрузить работу | POST  | /works                        |
| Получить отчёты  | GET   | /works/{assignmentId}/reports |
